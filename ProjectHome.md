# HowTO #

Windows Azure:

1. Download FreeConn Beta V0.1 Azure, unzip it.

2. Login to Azure portal

3. Deploy the package that unzip from the zip file.

4. Set your browser, point to YourAzureAccountName.cloudapp.net, port number is 32572
